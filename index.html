Practice modes:
Split — each audio segment is played; you repeat it twice.
Line — each full segment/line is played; you repeat it once.
Shloka — the entire shloka plays at once; you repeat it once.
Full sequence — for each shloka, runs all three phases in order: split (2 repeats each) → line (1 repeat each) → whole shloka (1 repeat).

How to use: upload one audio recording per shloka (named so the filename includes the shloka number). Choose a practice mode, set "Start from shloka #" if needed, then press Start. The current shloka's verses are always shown at the top.

Hear yourself: on Start, the app asks for microphone access. During each "Your turn" pause it records you and plays it back immediately so you can hear your own chanting.

Saved on this device: audio and text you upload are stored in this browser's local storage (IndexedDB) so they're available next time, without re-uploading.
